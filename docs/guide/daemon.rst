Daemon Management
=================

.. note::
   This page is under construction. Please contribute by adding content!

The daemon is the central orchestration service for MAPLE.

Starting the Daemon
-------------------

.. code-block:: bash

   maple serve

Options:

- ``--port`` - Specify custom port (default: 8080)
- ``--detach`` - Run in background

Checking Status
---------------

.. code-block:: bash

   maple status

Stopping the Daemon
-------------------

.. code-block:: bash

   maple stop
